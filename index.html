⚙️ Recommandations Techniques Webflow

🍪 Intégration Axeptio (RGPD) — CRITIQUE

Étape 1 — Créer le projet Axeptio
📍 axeptio.eu → compte → projet "re-set.club" → catégories : Nécessaires, Analytiques (GA4), Marketing (Pixel)
Étape 2 — Coller dans Webflow
📍 Project Settings → Custom Code → Head Code → coller EN PREMIER
<script>
window.axeptioSettings = {
  clientId: "VOTRE_CLIENT_ID",
  cookiesVersion: "re-set.club-fr"
};
(function(d,s){var t=d.getElementsByTagName(s)[0],
e=d.createElement(s);e.async=true;
e.src="//static.axept.io/sdk.js";
t.parentNode.insertBefore(e,t);
})(document,"script");
</script>
Étape 3 — Conditionner GA4
📍 Head Code → modifier le script GA4 existant
<script type="text/plain" data-cookieconsent="analytics"
  src="https://www.googletagmanager.com/gtag/js?id=G-XXXXX"></script>
<script type="text/plain" data-cookieconsent="analytics">
  window.dataLayer=window.dataLayer||[];
  function gtag(){dataLayer.push(arguments);}
  gtag('js',new Date());gtag('config','G-XXXXX');
</script>
Étape 4 — Conditionner Meta Pixel
📍 Même principe : type="text/plain" data-cookieconsent="marketing"
Étape 5 — Page /politique-cookies
📍 Créer Static Page → lister cookies → lien dans Footer Symbol

🏷️ SEO On-Page dans Webflow

SEO Title
📍 Page Settings → SEO Title (60 car. max)
Format : "Mot-clé — Complément | Re-Set Club"
Meta Description
📍 Page Settings → Meta Description (155 car. max)
Inclure mot-clé + CTA. Ex: "Découvrez notre sauna finlandais à Paris. Réservez en ligne."
Open Graph Image
📍 Page Settings → Open Graph Image (1200×630px)
H1 unique
📍 Designer → sélectionner texte → Tag = H1. Un seul par page.
Hiérarchie Hn
📍 Designer → Tag pour chaque texte. H1→H2→H3, ne jamais sauter.
Alt text images
📍 Designer → Image → Settings → Alt Text descriptif avec mot-clé
URL Slugs
📍 Page Settings → Slug court : sauna-paris (pas page-1)

📐 Templates de Pages Webflow

📐 Template : Page Service (Static Page)

Pour : /sauna-paris, /bain-froid-paris, /breathwork-paris

🔧 Webflow : Créer Static Page → slug → Page Settings SEO → construire sections empilées → Rich Text pour contenu → dupliquer pour autres services
Section 1
Hero Section
H1 avec mot-clé + sous-titre 2 lignes + CTA "Réserver" (Link Block vers booking) + image background
Section 2
Introduction (Rich Text)
200-300 mots. Qu'est-ce que [service], pourquoi chez Re-Set. Mot-clé dans les 100 premiers mots.
Section 3
Bénéfices (Grid)
4-6 bénéfices en cards (Div Blocks). Icône + H3 + texte court. Ex: "Récupération musculaire", "Boost immunitaire"
Section 4
Comment ça se passe (Rich Text)
Déroulé d'une séance étape par étape. H2 + liste numérotée. Rassure les nouveaux.
Section 5
Galerie (Grid d'images)
4-6 photos. Alt text optimisé sur chaque image.
Section 6
Tarifs aperçu + CTA
Résumé des prix + bouton "Voir tous les tarifs" → lien vers /tarifs
Section 7
FAQ (Rich Text)
5-8 questions. Utiliser H3 pour les questions. + Schema FAQ en Custom Code page.
Section 8
Cross-sell
"Découvrez aussi" → Link Blocks vers les 2 autres services. Maillage interne.
Section 9
CTA final
Bouton "Réserver maintenant" pleine largeur + lien vers booking.

📐 Template : Article Blog (CMS Collection)

🔧 Webflow CMS : Créer Collection "Blog Posts" → champs : Title, Slug, Thumbnail, Author, Body (Rich Text), Meta Description, Category → Template Page avec layout
Zone 1
Header article
H1 = titre (champ Title bindé). Date + temps de lecture + catégorie.
Zone 2
Image principale
Champ Thumbnail bindé. Alt text = titre article.
Zone 3
Corps (Rich Text bindé)
Champ Body. 1 200-1 500 mots. H2/H3 structurés. Liens internes vers pages services dans le texte.
Zone 4
CTA encadré
Div Block fixe dans le template (pas dans le Rich Text). "Envie de tester ? Réservez →" avec lien booking.
Zone 5
Articles liés
Collection List filtrée par catégorie (max 3). Maillage automatique entre articles.
