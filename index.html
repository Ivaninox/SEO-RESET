🔍 Audit rapide du site actuel

Vue d'ensemble

Re-Set est un lieu de sauna & bain froid au 5 rue Laffitte, Paris 9e. Site Webflow. Trois types de sessions proposées (Re-Boost, Re-Lax, Free-Flow) dans un concept "Social Wellness Club".

Trafic estimé : ~3 000 visites/mois, composé à 80-90% de recherches brandées ("re-set club", "reset club paris") et de trafic direct issu des Meta Ads. Le trafic SEO non-brandé — "sauna paris", "bain froid paris", etc. — est quasi inexistant. C'est tout le potentiel de croissance.

✅ Ce qui fonctionne

Meta description home

Bien rédigée : "Sauna & bain froid à Paris 9e. Sessions de thérapie de contraste guidées..."

Infos pratiques

Horaires dans le footer, adresse + lien Maps, téléphone. Bon signal local.

FAQ riche

9 questions-réponses détaillées couvrant les objections clés (température, déroulé, hygiène…).

CTA multiples

"Réserver une séance" présent à plusieurs endroits. Parcours de conversion clair.

Tonalité cohérente

Écriture premium, alignée avec le positionnement haut de gamme du lieu.

Données clés présentes

Sauna 50m², bains 3°C-10°C, 7 bains froids deux places. Mais ces infos sont cachées dans la FAQ.

🧩 Template de page service (Webflow)

Chaque page service (/sauna-paris, /bain-froid-paris, etc.) doit suivre cette structure :

Section 1 — Hero

H1 contenant le mot-clé principal + localisation. Sous-titre de 1-2 phrases. Bouton "Réserver". Image plein-écran du service en question.

Exemple H1 : "Sauna à Paris 9e — 50m² de sauna finlandais au cœur du 9ème"

Section 2 — Introduction

Paragraphe de 100-150 mots présentant le service. Contient le mot-clé principal + 2 variantes. Ton engageant, orienté bénéfice client.

Section 3 — L'espace / Le service

H2 type "Notre sauna finlandais de 50m²". Description physique, données chiffrées (température, dimensions, capacité). 2-3 photos avec alt text.

Section 4 — Les bienfaits

H2 type "Les bienfaits du sauna". Liste de 5-6 bienfaits avec icônes. 1-2 phrases par bienfait. Sources scientifiques si possible.

Section 5 — Déroulé d'une session

H2 type "Comment se déroule une session". Étapes numérotées (arrivée → vestiaire → session → repos). Rassure les débutants.

Section 6 — Nos sessions

H2 type "Nos sessions incluant le sauna". Présenter Re-Boost, Re-Lax, Free-Flow avec lien vers /tarifs et /reservation.

Section 7 — FAQ spécifique

H2 "Questions fréquentes". 4-5 questions propres à ce service (pas les mêmes que /faq). Balisage Schema FAQPage.

Section 8 — Avis / témoignages

2-3 avis clients liés à ce service. Nom + photo si possible. Renforce la confiance.

Section 9 — CTA final

Bloc de conversion : titre accrocheur + bouton "Réserver" + mention de l'adresse et des horaires. Lien vers /tarifs.

Template article blog (CMS Webflow)

Créer une Collection "Blog Posts" dans Webflow avec ces champs :

  • Name : Titre de l'article (H1 automatique)
  • Slug : URL de l'article
  • Main Image : Image à la une (og:image)
  • Body : Rich Text (contenu principal)
  • Excerpt : Résumé 155 caractères (meta description)
  • SEO Title : Plain text (title tag si différent du Name)
  • Category : Reference vers collection "Catégories"
  • Author : Plain text
  • Published Date : Date

Le template page doit inclure : H1 → date/auteur → image → corps → bloc "Articles liés" (3 articles) → CTA réservation.

⚙️ Recommandations techniques (Webflow)

Page Settings SEO (chaque page)

Dans Webflow, chaque page a un onglet Page Settings → SEO Settings. Pour chaque page :

  • Title Tag : remplir avec le title optimisé (50-60 caractères, mot-clé + "| Re-Set")
  • Meta Description : 150-155 caractères, mot-clé inclus, incitation au clic
  • Open Graph Title : identique au Title Tag
  • Open Graph Description : identique à la Meta Description
  • Open Graph Image : image 1200×630px propre à la page

Hiérarchie Hn — Comment corriger dans Webflow

  • Sélectionner chaque bloc texte → vérifier le tag dans le panneau de droite (Settings → Tag)
  • Règle : 1 seul H1 par page (le titre principal), puis H2 pour les sections, H3 pour les sous-sections
  • Ne jamais utiliser un H2 ou H3 juste pour le style → utiliser une classe CSS à la place
  • Erreur fréquente Webflow : les Headings par défaut sont H2. Changer le premier en H1 manuellement

Schema / Données structurées

Dans Webflow : Page Settings → Custom Code → coller dans le champ "Inside <head> tag".

Schema LocalBusiness (page Home)

Coller ce code dans le Custom Code de la page Home :

<script type="application/ld+json">
Objet @type: "HealthClub" avec name "Re-Set", address (5 rue Laffitte, 75009), geo coordinates, telephone, openingHoursSpecification pour chaque jour, url, image, priceRange "€€", sameAs [Instagram, site].
</script>

Je fournirai le JSON exact prêt à coller sur demande.

Schema FAQPage (pages FAQ et services)

Sur chaque page contenant une section FAQ, ajouter le Schema FAQPage dans le Custom Code avec toutes les questions/réponses de cette page.

Schema Organization (site-wide)

Dans Project Settings → Custom Code → Head Code (s'applique à tout le site). Inclut name, url, logo, contactPoint, sameAs.

RGPD / Bannière cookies

  • Outil recommandé : Axeptio (français, compatible Webflow, joli design)
  • Intégration : Créer un compte Axeptio → configurer les cookies (GA4, Meta Pixel) → copier le script → coller dans Project Settings → Custom Code → Footer Code
  • Conditionner les scripts : GA4 et Meta Pixel ne doivent se charger qu'après consentement. Axeptio gère ça nativement avec ses "vendors"
  • Créer la page /politique-cookies avec le widget Axeptio intégré
  • Ajouter le lien en footer : "Politique de cookies" + "Gérer mes préférences"

Sitemap & Google Search Console

  • Webflow génère automatiquement un sitemap à /sitemap.xml
  • Action : Aller sur Google Search Console → Ajouter la propriété re-set.club → vérifier via DNS ou balise HTML → Sitemaps → soumettre l'URL du sitemap
  • Dans GSC, demander l'indexation manuelle de chaque nouvelle page créée via "Inspection d'URL" → "Demander l'indexation"

Performance / Core Web Vitals

Score actuel estimé : 80 mobile / 85 desktop — Correct mais améliorable.

  • Images : Dans Webflow, activer le format WebP (Project Settings → le faire si dispo). Sinon, convertir manuellement avant upload. Max 200Ko par image.
  • Alt text : Chaque image → clic → Settings → remplir le champ Alt. Descriptif + mot-clé si naturel. Ex : "Sauna finlandais 50m² chez Re-Set Paris 9e"
  • Polices : Limiter à 2 polices max, 3-4 graisses. Utiliser les Google Fonts intégrées Webflow plutôt que des custom fonts.
  • Lazy loading : Webflow l'active par défaut sur les images en-dessous du fold. Vérifier que les images hero sont en "Eager" (chargement immédiat).

Redirections & erreurs

  • Si des pages sont renommées/supprimées → 301 redirect dans Project Settings → 301 Redirects
  • Créer une page 404 personnalisée avec liens vers Home, /tarifs, /reservation

🇬🇧 Page /homepage-en — Recommandations

La page anglaise existe mais manque d'éléments SEO. Voici ce qu'il faut ajouter :

Page Settings SEO

  • Title : Sauna & Cold Plunge in Paris 9th | Re-Set Club
  • Meta Description : "Experience contrast therapy in the heart of Paris. 50m² Finnish sauna, 7 cold plunge baths (3°C-10°C). Guided sessions. Book now at Re-Set."
  • OG Image : Même image que la home FR (1200×630px)

Contenu à ajouter sur la page

  • H1 : "Sauna & Cold Plunge in Paris 9th — Guided Contrast Therapy Sessions"
  • Section "The Experience" : Reprendre les 3 sessions (Re-Boost, Re-Lax, Free-Flow) traduites en anglais avec descriptions
  • Section "The Space" : Chiffres clés (50m² sauna, 7 cold baths, 3°C-10°C). 2-3 photos.
  • Section "How It Works" : Déroulé en 4 étapes (Arrive → Change → Session → Relax)
  • Section "FAQ" : 4-5 questions en anglais (What to bring, Temperature, Duration, First time tips)
  • Section "Location & Hours" : Adresse, carte, horaires, métro le plus proche
  • CTA : "Book a Session" vers /reservation

Hreflang (à ajouter dans Custom Code)

Dans le Custom Code "Inside <head>" de la page Home FR :

<link rel="alternate" hreflang="fr" href="https://www.re-set.club/" />
<link rel="alternate" hreflang="en" href="https://www.re-set.club/homepage-en" />

Dans le Custom Code de /homepage-en, ajouter les mêmes deux lignes (les deux pages doivent se référencer mutuellement).

🗓️ Roadmap Mois 1 & 2

🔴 MOIS 1 — Fondations

Semaine 1

Critique

Corriger le H1 de la Home : Dans Webflow, sélectionner le titre principal → Settings → Tag → H1. Contenu : "Sauna & Bain Froid à Paris 9e — Sessions de Thérapie de Contraste Guidées". S'assurer qu'il n'y a qu'un seul H1.

Critique

Corriger la hiérarchie Hn sur toutes les pages : Passer en revue chaque heading sur Home, /tarifs, /reservation. Corriger les tags (H2 pour sections, H3 pour sous-sections). Ne jamais sauter de niveau.

Critique

Remplir les Page Settings SEO : Sur chaque page existante, remplir Title, Meta Description, OG Title, OG Description, OG Image dans l'onglet Page Settings de Webflow.

Critique

Installer Axeptio : Créer compte → configurer GA4 + Meta Pixel comme vendors → coller le script dans Project Settings → Footer Code. Créer /politique-cookies.

Semaine 2

Critique

Schema LocalBusiness : Coller le JSON-LD dans Custom Code (head) de la Home.

Critique

Schema Organization : Coller dans Project Settings → Custom Code → Head (site-wide).

Critique

Google Search Console : Vérifier la propriété re-set.club → soumettre le sitemap.xml → demander l'indexation des pages existantes.

Fort

Optimiser les images : Vérifier format WebP, poids <200Ko, remplir tous les alt text. Vérifier lazy loading.

Semaine 3

Critique

Créer /sauna-paris : Nouvelle Static Page dans Webflow. Suivre le template 9 sections. Remplir Page Settings SEO. Ajouter Schema FAQPage. 800-1 200 mots.

Critique

Créer /bain-froid-paris : Dupliquer le layout de /sauna-paris → adapter tout le contenu. 800-1 200 mots.

Fort

Setup blog CMS : Créer la Collection "Blog Posts" dans Webflow avec les champs listés dans la section Templates. Construire le template de page article.

Semaine 4

Fort

Créer /breathwork-paris : 600-800 mots. Template service simplifié.

Fort

Enrichir /tarifs : Ajouter du contenu SEO (intro de 100 mots, comparatif des formules en texte, liens vers chaque page service).

Fort

2 premiers articles blog : Publier /blog/bienfaits-sauna + /blog/bienfaits-bain-froid. 1 200-1 500 mots chacun. Liens internes vers /sauna-paris et /bain-froid-paris.

Fort

Google Business Profile : Optimiser la fiche — catégories (Sauna, Centre de bien-être), horaires complets, photos du lieu, premier post GBP, commencer à demander des avis clients.

Moyen

Collecte d'avis Google : Mettre en place un process systématique — QR code en sortie de session, email post-visite avec lien direct vers la page d'avis GBP. Objectif : 10-15 avis à fin M2.
