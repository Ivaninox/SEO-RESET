Section 02

Diagnostic — Forces / Faiblesses / Opportunités

Analyse croisée des données GSC, GA4 et SEMrush.

Forces

1. Trafic organique en forte croissance

+848 % de trafic organique sur la période mesurée. Le site est encore jeune (Core Web Vitals actifs depuis mars 2026) et la courbe est ascendante, ce qui indique un potentiel SEO réel si les fondations techniques sont posées rapidement.

2. Tunnel de conversion fonctionnel

994 booking_success et 739 purchases pour 35 837 € de revenu sur la période. Le tunnel bsport est opérationnel. Le taux de rebond de la page /tarifs est à 20 %, signal d'un fort intérêt transactionnel.

3. Core Web Vitals au vert

4 URLs rapides mobile et desktop (GSC). Webflow livre des performances techniques solides sans optimisation spécifique. Ce n'est pas un frein au ranking — avantage rare à ce stade.

4. Backlinks éditoriaux de qualité

Malgré un AS de 8, 3 backlinks ont une vraie valeur : parisjetaime.com (référence institutionnelle Paris), gqmagazine.fr (DA élevé, audience masculine premium), inspire-paris.com. Pour un site de moins d'un an, c'est un signal éditorial fort.

5. Position #4 sur "bain froid paris"

Sur la requête commerciale locale la plus stratégique (140 vol., KD 10), Re-Set est déjà en page 1. Avec une page dédiée /bain-froid-paris, cette position peut passer #1–2 et générer un trafic qualifié significatif.

6. Concept différenciant dans un marché en croissance

Le "social club" autour du sauna et du bain froid est un positionnement clair. Le marché de la cold therapy est en forte croissance (bain de glace : 390 vol., cold plunge : +200 % YoY sur Google Trends). Re-Set arrive au bon moment.

Faiblesses

1. 99 % du trafic organique repose sur une seule requête de marque

Sur 901 visites organiques, 892 viennent de "re.se.t" (pos. 1). Toutes les autres requêtes génèrent moins de 10 visites combinées. Une seule page — la homepage — capte tout. Le site n'a aucune diversification sémantique.

2. Seulement 14 mots-clés rankés pour 5 pages indexées

La concurrence directe (Paris Ice Club) ranke sur 126 mots-clés. Meïso sur 933. Re-Set n'a aucune page de service dédiée, aucun article de blog, aucune page informationelle. L'architecture est trop plate pour absorber du trafic sémantique.

3. Architecture plate = cannibalisation

La homepage se positionne sur des requêtes aux intentions très différentes : "bain froid paris" (commercial local), "ice bath paris" (commercial EN), "sauna club paris" (commercial). Une seule page ne peut pas cibler efficacement ces intentions distinctes.

4. Authority Score très faible (8/100)

Sur les 130 backlinks (79 domaines), la quasi-totalité sont du spam : directories type "aged domains", "goooogla.com", "hebagh.cv". Seuls 3 backlinks ont une vraie valeur éditoriale. Sans netlinking actif, la progression sera lente sur les requêtes concurrentielles.

5. Pas de redirections 301 sur les variantes de domaine

Les 3 variantes (http://www., http://, https:// sans www) sont crawlées mais non redirigées. Les backlinks pointant vers ces variantes ne transfèrent pas leur autorité vers https://www.re-set.club/.

6. /old-home dans le sitemap

L'URL /old-home est encore listée dans sitemap.xml. Si cette page n'a plus d'utilité, elle dilue le budget de crawl et peut créer une confusion sémantique pour Google entre l'ancienne et la nouvelle homepage.

Plan d'action — Checklist

Tâches priorisées par impact × facilité. Cochez chaque tâche au fur et à mesure. Aucune unité temporelle — avancez à votre rythme.

🔧 Technique

Priorité haute

Constat : Aucun bandeau de consentement cookies détecté. GA4 se déclenche avant consentement — non conforme RGPD.

Action :

  • Créer un compte Axeptio sur axept.io et récupérer votre clientId
  • Dans Webflow → Project Settings → Custom Code → Head : coller le script Axeptio
  • Dans GTM : conditionner le tag GA4 au consentement Axeptio (trigger "Axeptio consent granted")
  • Vérifier dans Chrome DevTools (onglet Network) que GA4 ne se déclenche pas avant clic "Accepter"
<script> window.axeptioSettings = { clientId: "VOTRE_CLIENT_ID", cookiesVersion: "re-set.club-fr", }; (function(d, s) { var t = d.getElementsByTagName(s)[0], e = d.createElement(s); e.async = true; e.src = "//static.axept.io/sdk.js"; t.parentNode.insertBefore(e, t); })(document, "script"); </script>

Impact : Conformité RGPD. Évite les risques légaux (CNIL). Rassure les utilisateurs.

Priorité haute

Constat : 3 variantes crawlées par Google sans redirection : http://www.re-set.club/, http://re-set.club/, https://re-set.club/. Les backlinks pointant vers ces URLs ne transfèrent pas leur autorité.

Action : Dans Webflow → Hosting → Redirects, ajouter :

  • http://re-set.club/* → https://www.re-set.club/$1 (301)
  • http://www.re-set.club/* → https://www.re-set.club/$1 (301)
  • https://re-set.club/* → https://www.re-set.club/$1 (301)
  • Vérifier avec curl -I http://re-set.club que la réponse est bien 301

Impact : Consolidation du PageRank. L'autorité dispersée sur 3 variantes rejoint le domaine canonique.

Priorité haute

Constat : Aucune balise hreflang détectée. Google ne sait pas que / et /homepage-en sont des équivalents linguistiques. La page EN capte des impressions FR et vice-versa.

Action : Dans le Custom Code head de chaque page :

<!-- Sur / (homepage FR) --> <link rel="alternate" hreflang="fr" href="https://www.re-set.club/" /> <link rel="alternate" hreflang="en" href="https://www.re-set.club/homepage-en" /> <link rel="alternate" hreflang="x-default" href="https://www.re-set.club/" /> <!-- Sur /homepage-en --> <link rel="alternate" hreflang="fr" href="https://www.re-set.club/" /> <link rel="alternate" hreflang="en" href="https://www.re-set.club/homepage-en" /> <link rel="alternate" hreflang="x-default" href="https://www.re-set.club/" /> <!-- Même logique pour /tarifs ↔ /pricing et /reservation ↔ /booking -->

Impact : Google sert la bonne version linguistique. Le CTR EN devrait augmenter (actuellement 4,5 % sur /homepage-en).

Priorité moyenne

Constat : Le sitemap contient /old-home (obsolète) et ne contient pas de balises <lastmod>.

Action :

  • Dans Webflow → Pages : désactiver l'indexation de /old-home (toggle "Exclude from sitemap")
  • Vérifier que les nouvelles pages créées sont bien incluses dans le sitemap automatiquement
  • Dans GSC → Sitemaps : soumettre le sitemap mis à jour

Impact : Meilleure efficacité de crawl. Les nouvelles pages seront indexées plus rapidement.

Priorité moyenne

Constat : Aucun balisage structuré détecté. GBP fort (48 avis, 5 étoiles) mais le site ne renforce pas le signal local.

Action : Dans Webflow → Pages → Homepage → Custom Code (Before </body>) :

🔗 Netlinking — Développement de l'autorité

Standard

Constat : Sur 130 backlinks, la majorité provient de domaines spam (aged domains, goooogla.com, hebagh.cv…). Risque faible sur un jeune site mais à nettoyer.

Action :

  • Dans SEMrush → Backlinks → exporter la liste complète
  • Filtrer les domaines avec AS < 5 et ancre générique/irrelevante
  • Créer un fichier disavow.txt avec les domaines à ignorer
  • Soumettre dans GSC → Disavow Links Tool
  • Conserver : parisjetaime.com, gqmagazine.fr, inspire-paris.com

Impact : Signal de qualité de profil envoyé à Google. Potentiel +1–2 points d'AS sur 3 mois.
